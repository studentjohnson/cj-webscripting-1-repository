# FurMind Product Landing Page

## Description

FurMind is a responsive product landing page for a fictional AI-powered personality quiz web app.  
The app helps users discover their **outer and inner fursona** through a playful quiz and unlock an **AI companion chat experience**.

The design uses a cozy, dreamy aesthetic with **peach, gold, and soft brown** tones in light mode and **teal, purple, and midnight neon** tones in dark mode.

This project focuses on responsive layout, UI polish, accessibility, and theme switching.

## Features Checklist

### Layout
- CSS Grid used for major sections (features, pricing, creature gallery)
- Flexbox used for navbar, hero layout, and buttons
- Mobile-first design
- Two breakpoints (48rem and 64rem)

### Responsive Navigation
- Works on mobile and desktop
- Mobile toggle menu

### Animations
- Hover transitions on cards
- Hover transitions on buttons
- Fade-in animation on hero section

### Dark Mode
- CSS variables used for all colors
- Dark mode toggle button
- Theme persists using localStorage

### Accessibility
- One `h1` element
- Visible keyboard focus styles
- Readable color contrast
