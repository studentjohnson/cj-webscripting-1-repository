# Grand Line Logbook (Simple Fan Tribute)

A beginner-friendly multi-page site themed around **One Piece**. It demonstrates:
- Semantic landmarks (header, nav, main, footer)
- Accessibility basics (one h1 per page, headings in order, alt text, labeled form inputs)
- Responsive images and a responsive iframe embed
- A form with validation and a grouped option set (fieldset + legend)

## Pages
- `index.html`
- `about.html`
- `media.html`
- `contact.html`
